inside.icma.org

This intranet features a very flexible, database-driven navigation system that can be easily updated to accommodate new features or applications. Each application is color-coded, to provide a distinct visual identity to each component while retaining a consistent graphic look throughout the site.

The entire site is done in ASP/Visual Basic, with JavaScript used for field validation. The underlying database tables reside on a Microsoft SQL Server.

Sample screen shots of some of the intranet's features follow:

Equipment Checkout/Resource Management

This feature allows ICMA staff to reserve and check out resources such as laptops and projectors. Administrators can maintain a very flexible database of resources, monitor and approve resource reservations, and check equipment in and out.

 

Staff Travel & Leave Notice Board

This application allows ICMA staff to post travel and leave notices, along with contact information during their absence.

 

 

Membership Database & Label Generator

The ICMA intranet now provides easy access to the association's membership database. One of the most useful features of the application allows staff to generate standard-format mailing labels from this database; the system creates a Word document on the fly according to the criteria specified by the user.
